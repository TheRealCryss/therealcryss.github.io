Wetterbericht

[Stable 1.0 for Weather Report 1.042]

Wetterbericht ist der deutsche Übersetzungspatch von Weather Report, ein Fallout 4 Mod von Greslin. Dieses wird benötigt.

Beschreibung:

Für eine detailierte Beschreibung besuche bitte die Github-Seite.

Installation:

Platziere einfach den Patch nach dem Weather Report mod in deinem Mod-Manager (am besten MO2).

Voraussetzungen:

  • Weather Report
    (Mod von Greslin. Das gibt es auf seinem Discord)

Bekannte Inkompatibilitäten:

  • nichts bekannt
